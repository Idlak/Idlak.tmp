Detailed Description

Base class for all objects which require voice data.

The class wraps expat and has virtual fuctions to handle the XML parsing. In the objects derived from this, a file will typically be built into a bespoke structure for rapid processing. The callback functions need to be static to be set in expat, these then call the virtual functions.
