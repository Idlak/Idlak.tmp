Detailed Description

The trules object contains regular expression and lookup tables.

Example of a regex which finds items like 39°

  <regex name="int_degree">
      <exp>
          <![CDATA[^([0-9]+)(°)$]]>
      </exp>
  </regex>

Example of a lookup table which converts telphone prefixes into the word telephone 

<lookup name="telephone_prefix_table">
  <exp>
    <![CDATA[{"tel":"telephone", "telephone":"telephone", "ph":"phone", "t":"telephone", "phone":"phone"}]]>
  </exp>
</lookup>
